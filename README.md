# QueueLess 🚀

**Skip the Wait, Not the Visit.**

A Smart Queue Management Platform designed for clinics, salons, banks, educational institutions, and government offices. QueueLess enables users to book appointments, track live queues, receive notifications, and reduce waiting time through intelligent queue management.

---

## 📌 Project Overview

QueueLess consists of two major modules:

### 📱 Android Application

A Flutter-based mobile application providing appointment booking, queue tracking, payments, AI assistance, and business management features.

### 🌐 Web Application

A modern web-based platform providing queue management, appointment booking, business dashboards, analytics, and real-time queue monitoring.
